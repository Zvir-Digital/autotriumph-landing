# AutoTriumph Landing

Landing page для AutoTriumph.

## Технологии

- Next.js 16
- TypeScript
- Tailwind CSS
- pnpm

## Разработка

Запуск dev-сервера:

```bash
pnpm dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка

```bash
pnpm build
pnpm start
```

## Линтинг

```bash
pnpm lint
```
